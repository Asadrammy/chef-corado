DEPLOYMENT INSTRUCTIONS
================================================================================

1. PREPARE INFRASTRUCTURE
   - PostgreSQL 13+ with SSL
   - Node.js 18+ LTS
   - PM2 process manager
   - Nginx reverse proxy
   - SSL certificate (Let's Encrypt)

2. DEPLOY APPLICATION
   - Clone repository
   - Install dependencies: npm install --production
   - Build: npm run build
   - Start: pm2 start npm --name "chef-marketplace" -- start

3. CONFIGURE NGINX
   - Copy config from deployment guide
   - Test: sudo nginx -t
   - Enable: sudo systemctl enable nginx
   - Start: sudo systemctl start nginx

4. ENABLE SSL
   - Certbot: sudo certbot certonly --nginx -d yourdomain.com
   - Auto-renewal: sudo systemctl enable certbot.timer

5. CONFIGURE BACKUPS
   - Create backup script
   - Schedule with cron: 0 2 * * *
   - Test restoration

6. CONFIGURE MONITORING
   - Set NEXT_PUBLIC_SENTRY_DSN
   - Configure Logtail/Datadog
   - Set up alert rules
